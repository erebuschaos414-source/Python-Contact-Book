# Python Contact Book

A command-line contact manager built in Python. Contacts persist between 
sessions using local JSON file storage.

## Features
- Add contacts with name, phone, and optional email
- Search contacts by name
- List all saved contacts
- Delete contacts
- Auto-saves to a local JSON file

## How to run

Make sure you have Python 3.10+ installed, then:

```bash
python main.py
```
